# 扑克牌游戏

一个基于 React + Vite + Zustand + Supabase 的多人在线扑克牌游戏。

## 🎮 项目特点

- **单页应用**: 使用 100vw × 100vh 的全屏设计，无需滚动
- **赛博朋克风格**: 霓虹色调、几何形状、发光效果、故障艺术
- **实时同步**: 基于 Supabase Realtime 的多人实时对战
- **状态管理**: 使用 Zustand 进行简洁的状态管理
- **响应式设计**: 支持桌面端和移动端

## 🎨 设计风格

### 复古未来主义 + 赛博朋克美学
- **配色**: 霓虹粉 (#ff006e) + 电光蓝 (#00f0ff) + 赛博紫 (#8b00ff)
- **字体**: Orbitron (标题) + Rajdhani (正文)
- **特效**: 扫描线、霓虹光晕、故障动画、渐变边框
- **设计**: 斜角裁切、几何形状、网格背景

详细设计说明请查看 [赛博朋克设计说明.md](./赛博朋克设计说明.md)

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

项目中的 `.env` 文件已配置好 Supabase 连接信息。

### 3. 确保 Supabase 数据库已设置

请参考项目中的 `SUPABASE_SETUP.md` 文档，确保已经创建了必要的数据库表。

### 4. 启动项目

```bash
npm run dev
```

访问 `http://localhost:3000` 开始游戏。

## 🛠️ 技术栈

- **前端框架**: React 19
- **构建工具**: Vite 7
- **状态管理**: Zustand 5
- **数据库**: Supabase (PostgreSQL)
- **实时通信**: Supabase Realtime
- **样式**: CSS (使用 CSS 变量和动画)

## 📝 开发说明

### 添加新功能

1. **常量**: 在 `src/constants/` 中定义
2. **状态**: 在 `src/stores/gameStore.js` 中管理
3. **工具函数**: 在 `src/utils/` 中实现
4. **UI 组件**: 在 `src/components/` 中创建

### 样式规范

- 使用 `vh` 和 `vw` 单位确保全屏适配
- 保持统一的圆角 (`border-radius`) 和阴影 (`box-shadow`) 样式
- 使用 CSS 动画增强交互体验
- 响应式断点: 768px (移动端)
